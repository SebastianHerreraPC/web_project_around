# Mi Proyecto de Viajes

Este es un proyecto web simple para mostrar información sobre lugares de interés, con funcionalidades básicas de perfil de usuario.

## Descripción

Este proyecto consiste en una página web que muestra lugares de interés y permite a los usuarios ver información sobre estos lugares. Además, cuenta con funcionalidades de perfil de usuario que permiten editar el nombre y el trabajo del usuario.

## Instrucciones de Uso

1. Abre el archivo `index.html` en tu navegador web.
2. Explora los lugares de interés en la sección de tarjetas.
3. Haz clic en el botón "Editar el perfil" para modificar tu nombre y trabajo en la sección de perfil.

## Estructura del Proyecto

- **index.html**: Página principal del proyecto.
- **index.css**: Hoja de estilo que define la apariencia de la página.
- **index.js**: Archivo JavaScript para la lógica interactiva en la página.
- **/images**: Carpeta que contiene las imágenes utilizadas en la página.

## Personalizaciones

- **Edición del Perfil**: El formulario de edición de perfil está oculto inicialmente y se muestra al hacer clic en "Editar el perfil".
- **Valores Predeterminados**: Los campos del formulario de edición de perfil tienen valores predeterminados que pueden personalizarse.
- **Desbordamiento de Texto**: Se ha aplicado un estilo para controlar el desbordamiento de texto en los campos de nombre y trabajo en el perfil.

## Tecnologías Utilizadas

- HTML
- CSS
- JavaScript
- Figma (para diseño y prototipado)
